Add to bookmarkd: Bookmarklet

Drag the button below to your bookmarks bar.
Then, click it on any site you want to save to bookmarkd — the title and URL will be auto-filled!

Add to bookmarkd
How to use:
  1. Click and drag the Add to bookmarkd button above to your bookmarks bar. If you don't see the bookmarks bar, press Ctrl+Shift+B (Windows) or Cmd+Shift+B (Mac) to show it.
  2. When you're on a web page you want to save, just click your new bookmarklet. It will open a small popup window that adds the bookmark to bookmarkd.
  3. Safari users: If you get a popup blocked message, allow popups for the site you're bookmarking, then try again.

Note: The bookmarklet is currently configured for http://localhost:8080. If your bookmarkd runs on a different port or domain, you'll need to edit the bookmarklet code after dragging it to your bookmarks bar.
Right-click the bookmark → Edit → Update the URL in the bookmarklet code where it says bookmarkdURL='http://localhost:8080'
